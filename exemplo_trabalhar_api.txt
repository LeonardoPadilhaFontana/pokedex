Requisição HTTP para API/NORMAL

URL: CAMINHO PARA CHEGAR NA API. EX:(HTTPS://POKEAPI.CO/) PATH = CAMINHO DE IDENTIFICACAO DO RECURSO
= API/V2/POKEMON

REQUEST METHOD: GET / PUT / POST / DELETE

REQUEST HEADER -> CONFIGURAÇÃO ENVIADA PELO NAVEGADOR QUE FAZ A SOLICITACAO

RESPONSE HEADER -> CONFIGURAÇÃO DEVOLVIDA PELO SERVIDOR SOLICITADO PELA API

REQUEST BODY -> EXEMPLO DE BODY COM JSON
{
	"NAME": "TESTE"
}


STATUS CODE: 	200 - 299 -SUCESSO
		300 - 399 - REDIRECIONAMENTO
		400 - 499 - ERRO DO CLIENTE
		500 - 599 - ERRO INTERNO DO SERVIDOR

Docs: https://http.cat/
